Object-oriented programming (OOP)

All R statements where you create objects – “assignments” – have this form:

objectName <- value

  • You provide your object with information or “functions” (numbers, characters, giant matrices, etc.) using the operator “<-”. We call the process “assignment”. Why not “=”?

  • We can use a shortcut! … just press "alt" "-".

Example

x <- 5

y <- sin(5 - pi/2)

z <- seq(from = 1, to = 100, by = 1)

This_is_cool <- “Hello”

and_this <- no_way

# construct constants
a <- 2

# draw 40 uniformly random numbers
x <- runif(40)

# Build a function
y <- a + b * x + rnorm(40, sd = sqrt(sig_sq))

# Average of our uniformly random variable (x)
